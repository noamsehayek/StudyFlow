# StudyFlow – Android App

StudyFlow is a simple Android application built in Kotlin that helps students manage their daily study tasks.

## Features
- Home screen displaying study tasks using RecyclerView
- Add Task screen to add new tasks (Assignment / Exam)
- Mark tasks as completed using checkboxes
- Clean and simple UI
- No database or API usage (in-memory data)

## Technologies Used
- Kotlin
- Android Studio
- XML layouts
- RecyclerView
- Intents for navigation

## Project Context
This app was developed as part of the **Mobile Apps Development – Final Project**.
